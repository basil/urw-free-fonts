# URW Free Fonts

## Introduction

URW released several free fonts [for use with TeX](https://ctan.org/tex-archive/fonts/urw).
These original files are still available and can be freely downloaded, but they can be hard to find.
So I tracked them down and wrote a script to convert them into aÂ new package of OTFs, TTFs, and webfonts.

## Prerequisites

- [FontForge](https://fontforge.org)

## Build

```bash
$ ./build.sh
$ ls dist/*.zip
dist/A030.zip
dist/GaramondNo8.zip
dist/LetterGothic.zip
dist/URWAntiquaT.zip
dist/URWClassico.zip
dist/URWGroteskT.zip
```
